As of 2026-03-26, current priorities are:

1. **Authenticated login for Instagram and TikTok** — move beyond anonymous browser interception to logged-in sessions that unlock richer API responses (more data, fewer blocks, access to feeds/following content)
2. **Pull more resources** — extract more data from each platform interaction (deeper comment threads, related content, user profiles)
3. **Analyze and synthesize** — improve the analysis pipeline to turn raw social data into structured, actionable insights
4. **Agent-actionable outputs** — ensure insights are formatted so that downstream agents can directly act on them (not just informational summaries)

**Why:** The goal is to close the loop from "social listening" to "social action" — agents should be able to discover signal, understand it, and take steps based on it.

**How to apply:** When making implementation decisions, prioritize auth flows and data richness over cosmetic improvements. The analysis pipeline is the key differentiator.
