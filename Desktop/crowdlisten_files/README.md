# CrowdListen Files — Directory Map

Monorepo workspace for the CrowdListen audience-insight platform.

| Directory | Status | Description |
|-----------|--------|-------------|
| `crowdlisten_deployed/` | **Active** | Production codebase — frontend (React/Vite on Vercel), backend services, content-core |
| `crowdlisten_agent/` | **Active** | Agent service — Agno-based analysis & writing pipeline, deployed on EC2/Docker |
| `crowdlisten_mcp/` | **Active** | MCP server for CrowdListen social media retrieval |
| `crowdlisten_prototyping/` | **Active** | Research prototypes — narrative report generator, TikTok MCP, experiments |
| `crowdlisten_archived/` | **Archived** | Previous iterations (aibrary-fe, aibrary-backend, coaching-llms, skills, etc.) |
| `crowdlisten_costs/` | **Reference** | Cost tracking and analysis |
